# Датасет: Судебные акты по ст. 14.13 КоАП РФ (арбитражные управляющие)

Размеченные решения арбитражных судов по делам о привлечении арбитражных управляющих к административной ответственности.

## Содержание репозитория

```
├── data/
│   ├── dataset.jsonl        # Все записи (JSONL, 1 запись = 1 строка)
│   └── dataset.csv          # Плоская таблица (массивы сериализованы через " | ")
├── raw/
│   └── *.json               # Исходные размеченные JSON-файлы (по одному на акт)
└── README.md
```

## Описание полей

### Идентификация дела

| Поле | Тип | Описание |
|---|---|---|
| `case_number` | string | Номер дела (напр. `А40-12345/2024`) |
| `document_date` | string (ISO 8601) | Дата судебного акта |
| `court_name` | string | Название суда |
| `judicial_instance` | string | Инстанция: `апелляция`, `кассация` |
| `judicial_circuit` | string | Судебный округ |

### Арбитражный управляющий

| Поле | Тип | Описание |
|---|---|---|
| `au_name` | string | ФИО арбитражного управляющего |
| `au_sro` | string \| null | Саморегулируемая организация АУ |
| `bankruptcy_procedure` | string | Процедура банкротства (конкурсное производство, наблюдение и др.) |

### Должник

| Поле | Тип | Описание |
|---|---|---|
| `debtor_name` | string | Наименование должника |
| `debtor_category` | string | `организация`, `гражданин`, `ИП` |

### Административное дело

| Поле | Тип | Описание |
|---|---|---|
| `initiating_authority` | string | Орган, составивший протокол (Росреестр, прокуратура и др.) |
| `complaint_source` | string | Источник жалобы: `кредитор`, `должник`, `собственник имущества должника`, `иное лицо`, `собственная инициатива органа`, `null` |
| `dispute_subject` | string | Предмет спора: `привлечение_АУ`, `оспаривание_акта_органа`, `иное` |
| `koap_article` | string | Квалификация (ч. 3 / ч. 3.1 ст. 14.13 КоАП РФ) |
| `bankruptcy_law_violations` | array[string] | Нормы Закона о банкротстве, которые нарушил АУ |
| `violation_category` | array[string] | Категории нарушений (инвентаризация, собрание_кредиторов и др.) |

### Описание нарушений

| Поле | Тип | Описание |
|---|---|---|
| `violation_description` | string | Подробное изложение вменяемых нарушений (разметчик) |
| `violation_description_cited` | string | Цитаты из текста судебного акта о нарушениях |
| `violation_summary` | string | Краткое резюме нарушений (2–3 предложения) для RAG-поиска |

### Исход дела

| Поле | Тип | Описание |
|---|---|---|
| `outcome` | string | `привлечён` / `не привлечён` / `направлено на новое рассмотрение` / `постановление_отменено` |
| `penalty_type` | string | `штраф` / `дисквалификация` / `предупреждение` / `null` (если не привлечён) |
| `disqualification_term_months` | int \| null | Срок дисквалификации в месяцах |
| `penalty_amount_rub` | int \| null | Размер штрафа в рублях |
| `non_liability_ground` | string | Основание освобождения: `малозначительность`, `отсутствие состава`, `процессуальные нарушения`, `законный отказ`, `н/п` |
| `de_minimis_applied` | bool | Применена ли малозначительность (ст. 2.9 КоАП РФ) |

### Аргументация

| Поле | Тип | Описание |
|---|---|---|
| `court_reasoning` | string | Мотивировка суда |
| `au_defense_arguments` | string | Доводы защиты АУ |
| `precedents_cited` | array[string] | Цитируемые прецеденты и разъяснения ВАС/ВС РФ |

### Оценка качества

| Поле | Тип | Описание |
|---|---|---|
| `value_score` | int (0–10) | Оценка ценности дела для датасета (чем выше — тем ценнее) |
| `value_reason` | string | Обоснование оценки |
| `_notes` | string | Примечания разметчика |
| `_source_file` | string | Имя исходного файла |

## Использование

### Python (JSONL)
```python
import json

with open("data/dataset.jsonl", encoding="utf-8") as f:
    records = [json.loads(line) for line in f]

# Только дела с дисквалификацией
disqualified = [r for r in records if r["penalty_type"] == "дисквалификация"]

# Только высокоценные дела
top = [r for r in records if (r.get("value_score") or 0) >= 7]
```

### Pandas (CSV)
```python
import pandas as pd

df = pd.read_csv("data/dataset.csv")

# Распаковать массив категорий нарушений
df["violation_category__list"].str.split(" | ")
```
